DATA Acquisition README
=======================

Modules
-------

  common   : Common utility functions and configuration.
  geo      : SHP to Postgres Spatial data importer and indexing scripts.
  osm      : OSM (Open Street Map) dump to PostGres importer scripts.
  rdf      : RDF file to Virtuoso importer scripts.
  scrapers : All SW scrapers witten for data acquisition.
  socrata  : Socrata JSON API to PostGres importer scripts.
  csv      : CSV, XLS to PostGres importer scripts.
  sw       : ScraperWiki JSON API to PostGres importer scripts.
  tools    : CLI tools used by various modules.
