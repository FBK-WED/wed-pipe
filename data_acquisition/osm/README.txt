OSM README
==========

Description
-----------

  Set of scripts for ingesting OSM (Open Street Map) data into PostGres.

Usage
-----
 
  $ ./ingest-data.sh -d wedpipe-test -s osm -a 'http://download.gfoss.it/osm/osm/regioni/trentino-alto_adige.osm.bz2'

  For more help:

  $ ./ingest-data.sh --help
